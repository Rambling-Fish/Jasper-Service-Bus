prerequisites
-------------
1. ensure you have activemq in your local maven repo

Build Instructions
------------------
1. from top most directory run
    a. mvn clean install
4. All the jApp zips will be in the newly created target folder
4. The jCore.zip will also be in the target folder

Setup Instructions
----------------
1. unzip the jCore.zip
2. cd to unzipped folder, jasper-engine-1.0
3. run setup : ./setup.sh

Starting and Stoping Jasper
---------------------------
use the jasper.sh script, examples
  - ./jasper.sh start
  - ./jasper.sh status
  - ./jasper.sh stop

Deploy, Configure and Undeploy jApps
------------
1. copy jApp zips into the jasper-engine-1.0/jApps folder
2. run jAppManagement.sh script and follow menu screens.
